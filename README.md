# 俄罗斯方块游戏

一个基于 HTML5 Canvas 的俄罗斯方块游戏。

## 游戏特点

- 经典的俄罗斯方块玩法
- 支持键盘控制
- 分数系统
- 等级系统
- 下一个方块预览
- 暂停功能

## 如何游玩

1. 使用方向键控制方块：
   - ← → : 左右移动
   - ↑ : 旋转
   - ↓ : 加速下落
   - 空格 : 直接落下

2. 点击"开始游戏"按钮开始
3. 可以随时点击"暂停"按钮暂停游戏
